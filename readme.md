STM Events Website - Development Scope
1. Project Overview
Client: STM Events
Industry: Nightlife / Event Production
Purpose of website:
Promote STM Events brand


Showcase nightlife events and productions


Display services offered by STM Events


Redirect users to Eventbrite for ticket purchases


Showcase equipment rental offerings


Present Resident DJs


Build credibility for event production business


Website style:
Modern nightlife / underground club aesthetic


Inspired by Boiler Room style and Lab54 website


Dark minimal design with strong visuals









2. Website Pages / Structure
1. Home Page
Goal: Strong visual first impression.
Sections:
Hero section with video background or event footage
Highlight next upcoming event
Event countdown timer
Quick overview of STM Events
Visual preview of gallery / events
Navigation to other sections



2. Events & Tickets Page
Events Listing Page
Shows:
List of all upcoming events
Each event card links to its individual event page


Individual Event Page
Each event page must contain:
Event title
Event date
Location
Lineup
Images / videos
Event description
Eventbrite ticket button
Events are expected:: Around 10–15 events per year
Events should be created from the admin panel and automatically generate a new event page.
3. Gallery / Art Page
Media showcase of:
Event photos
Aftermovies
DJ sets
Event videos
Trailer / rebrand videos
Media must be uploadable via admin panel.

4. About Page
Content:
Introduction to STM Events
What the brand does
Event production overview

5. Services Page
Showcase services offered by STM Events:
Services include
DJ bookings
Private events
Club events
Event production


Each service should have:
Short description
Visuals if available

6. Equipment Rental Page
Rental listings for:
Sound systems
DJ equipment
Lighting equipment
Initial listings expected:
5–10 equipment items


Each listing should include:
Equipment name
Images
Description
Price
Rental process:
No direct online booking
Users must contact via email or phone
Equipment listings must be editable from admin panel.

7. Resident DJs Section
Showcase STM Events Resident DJs.
Each DJ profile should include:
DJ name
DJ logo
Photo

8. Contact Page
Features:
Contact form

Equipment rental

Event bookings
